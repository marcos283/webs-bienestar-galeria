# Galería de webs de bienestar

Repositorio estático para publicar y comparar prototipos web navegables.

## Demos incluidas

- Espacio Senda — Casa luminosa (Stitch original importado)
- Método Clara — Pilates claro
- Espacio Senda — Casa luminosa HTML
- Espacio Senda — Guía práctica HTML

## Uso

Abrir `index.html` o publicar la raíz del repo en GitHub Pages / Netlify.

Flujo recomendado:

1. Generar direcciones visuales en Stitch.
2. Importar material bruto sin tocarlo.
3. Publicar una demo navegable en esta galería.
4. Convertir la ganadora en HTML/CSS limpio para producción.
